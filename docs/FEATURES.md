# 完整功能清单

这个清单用于防止“现代化改版”时遗漏旧 Reader 已有能力。完成状态以实际可用和有验证为准，而不是以页面上出现入口为准。

## 核心阅读

- [x] 响应式书架与继续阅读
- [x] 滚动/分页阅读、章节切换、目录、主题、字体、字号、行高、版心
- [x] 服务端进度接口适配
- [x] 键盘、左右触控热区、横向滑动翻页与阅读界面隐藏
- [x] 书签位置同步、章节内查找、正文替换/正则净化、按需简繁转换
- [x] 自动阅读与系统 TTS
- [x] 章节预加载、IndexedDB 整本离线下载/进度/取消/删除、服务端整本缓存入口
- [x] 完全断网冷启动：离线重开 PWA、恢复书架、目录、正文和本机阅读进度
- [x] 音频源播放与漫画源长图阅读（媒体统一走鉴权代理）
- [x] 明确不做仿真/Kindle 模式与笔记编辑（个人纯阅读场景主动精简）
- [ ] HTTP TTS（当前使用设备系统 TTS，不常驻服务端语音任务）

## 书源与发现

- [x] Reader / Legado JSON 导入与服务端保存
- [x] 单/多源搜索 API 适配
- [x] HTTP / Cookie / 登录-WebView 状态表达
- [x] 书源基础字段 + 完整 JSON 编辑、导入导出、单/批量启停、排序、批量检测、失效治理与 SSE 调试日志
- [x] 聚合书海/分类发现、服务端去重与游标下一页；默认关闭，只在用户开启并进入时请求
- [x] 换源、源可用性检测、代理登录 UI 与 Cookie 回传
- [x] 可选 WebView Sidecar 执行、私网拦截、资源瘦身和空闲回收
- [x] 常用请求/搜索/发现/详情/目录/正文规则的折叠表单，同时保留完整 JSON 与未知字段
- [ ] 鼠标拖动排序（当前使用明确的上移/下移，手机和键盘同样可用）

## 资料与维护

- [x] 书籍分组与移动
- [x] TXT / EPUB / MOBI / PDF 服务端上传与解析
- [x] TXT / EPUB / MOBI / PDF 原文件导出与本地书仓
- [ ] UMD 解析与 TXT 自定义目录规则 UI
- [x] RSS 源新增、完整规则导入导出、文章列表与阅读
- [x] 替换规则管理
- [x] WebDAV 文件上传、列表、下载和删除；全量配置备份与恢复
- [x] 少量用户、本地书/WebDAV 权限与管理员视图（AI 权限不开放）
- [x] 密码修改、管理员重置密码与用户删除 UI
- [x] 手动增量更新书架、失败源筛选/批量删除与容器健康检查
- [ ] 可配置的定时更新与自动升级（避免少用户部署无意义常驻轮询）
